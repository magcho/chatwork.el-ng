# chatwork.el-ng
Chatwork client for emacs-ng.


## how to use
```lisp
(setq chatwork-ng-token "XXXXXXXXXXXXXXXX")
(eval-js "import 'https://raw.githubusercontent.com/magcho/chatwork.el-ng/main/chatwork-el-ng.ts'")
```

`M-x chatwork-ng-send-message`
